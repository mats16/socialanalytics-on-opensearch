# Twitter Streaming Reader

### about Twitter Authentication

- `-e CONSUMER_KEY=...`
- `-e CONSUMER_SECRET=...`

### Other options

- `-e TWITTER_TOPICS=AWS,EC2,RDS`
- `-e TWITTER_LANGUAGES=en,ja`
- `-e TWITTER_FILTER_LEVEL=none`  # default: none
